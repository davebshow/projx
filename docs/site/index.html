projx - Graph transformations in Python.

projx provides a simple and extensible API for interacting with graphs in Python. It's core functionality is built around making graph transformations in memory using the NetworkX module and a DSL based on Neo4j's Cypher query language. It also provides an extensible JSON based ETL pipeline (roughly modeled after orientdb-etl) that allows you to easily translate graph data between various persistent and in-memory data sources.

For a quick tour, check out the getting started.
